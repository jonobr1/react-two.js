## Quick Start

```jsx
import { Canvas, Circle, useFrame } from 'react-two.js'
import { useRef } from 'react'

function RotatingCircle() {
  const ref = useRef()
  useFrame((t) => ref.current.rotation = t * 0.5)
  return <Circle ref={ref} radius={50} fill="#00AEFF" />
}

function App() {
  return (
    <Canvas width={800} height={600}>
      <RotatingCircle />
    </Canvas>
  )
}
```

## Component Categories

### Core Components
- **Canvas** - Main container component that creates the Two.js instance
- **Group** - Container for organizing and transforming multiple shapes

### Primitive Shapes
- Circle, Rectangle, RoundedRectangle
- Ellipse, Line, Polygon, Star
- ArcSegment
- Text

### Advanced Components
- Path, Points
- LinearGradient, RadialGradient
- Texture, Sprite, ImageSequence
